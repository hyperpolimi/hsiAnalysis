# hsiAnalysis
MATLAB scripts for the analysis of datasets acquired with hyperspectral camera or microscope based on common-path, ultra-stable, birefringent interferometer TWINS.  
Instructions for installation and for codes are available in [Instructions](https://polimi365-my.sharepoint.com/:f:/r/personal/10674735_polimi_it/Documents/PhD/Lab.%20Hyperium/Analysis%20Software?csf=1&web=1&e=q8oDFT).
