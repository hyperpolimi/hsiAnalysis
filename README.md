# hsiAnalysis
MATLAB scripts for the analysis of datasets acquired with hyperspectral camera or microscope based on common-path, ultra-stable, birefringent interferometer TWINS.  
Instructions for installation and for codes are available in [Instructions](https://polimi365-my.sharepoint.com/:f:/g/personal/10674735_polimi_it/IgCI0fZ1sPBGQa5LRTMyHeD6ARrs6ayLg3ujGntV3Hl22YY?e=nwSFY5).
